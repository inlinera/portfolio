Javascript

JavaScript is a high-level interpreted programming language that is primarily used as a scripting language for web development. It is one of the three main technologies of the World Wide Web, along with HTML and CSS.
The JavaScript programming language allows you to create interactive web pages and is an integral part of web applications. While HTML defines the structure and layout of a web page, and CSS gives it style, JavaScript makes it interactive by providing dynamic content and user interaction.

Css

CSS (Cascading Style Sheets) is a language for describing the appearance of an HTML document. This is one of the basic technologies of the modern Internet. Almost no website can function without CSS, which is why HTML and CSS work together.
Cascading Style Sheets work with HTML, but it's a completely different language. HTML structures the document and organizes information, and CSS interacts with the browser to give the document its appearance

Html

HTML (Hypertext Markup Language) is a hypertext markup language. This is necessary to place elements on a web page: text, images, tables and videos.
When you visit a website, the browser downloads an HTML file containing information about the structure and content of the web page. HTML, as it were, builds the visual basis of the site, but does not “launch” the site on its own. It simply specifies where the elements are located, what their basic design will be, and where the elements' styles and scripts will come from

Git

Git is a version control system. Git is a distributed version control system, which means that it does not have a single central server and allows multiple developers to work together without interfering with each other. Git also has features like branching, tagging, and commit history that help developers manage changes to their code and see who made what changes.
